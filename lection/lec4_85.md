* Компилируем сервер:
    ```shell
    javac Server
    ```
* Компилируем клиент:
    ```shell
    javac Client
    ```
* Запускаем сервер:
    ```shell
    java Server
    ```

* Дождавшись надписи "server is started", запускаем любое количество клиентов:
    ```shell
    java Client test1
    java Client test2
    ...
    java Client testN
    ```
